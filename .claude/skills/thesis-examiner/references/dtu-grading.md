# DTU thesis assessment & the Danish 7-point scale

Use this to frame an **indicative, advisory** grade band — not a verdict. The
official grade is decided by the examiner and external censor against the
course/thesis learning objectives. State clearly that your band is a starting
point for the examiner's own judgment.

## The Danish 7-point grading scale (7-trinsskala)

Grades, from top, with the standard descriptor (the "degree of fulfilment of the
learning objectives, with respect to weaknesses"):

| Grade | ECTS | Meaning |
|-------|------|---------|
| **12** | A | Excellent — comprehensive fulfilment of the objectives, with **no or few insignificant weaknesses**. |
| **10** | B | Very good — comprehensive fulfilment, with **a few minor weaknesses**. |
| **7**  | C | Good — fulfilment with **a number of weaknesses**. |
| **4**  | D | Fair — minimal acceptable fulfilment, with **a number of significant weaknesses**. |
| **02** | E | Adequate — the **minimum** acceptable fulfilment (pass threshold). |
| **00** | Fx | Inadequate — does not meet the minimum (fail). |
| **-3** | F | Unacceptable — entirely unacceptable performance (fail). |

Note the gaps: there is no 11, 9, 8, 6, 5, 3, or 1. **02 is the lowest passing
grade.** When you give a band, name an actual grade or pair (e.g. "10–12 (B–A)"
or "7 (C)"), not an out-of-scale number.

## How DTU thesis assessment works (context)

- A DTU MSc thesis is typically **30 ECTS** (sometimes 32.5/35/45/60 depending on
  programme); roughly a half-year to full-year of work. Calibrate scope
  expectations to the ECTS load if you know it.
- Assessment is by the **supervisor (examiner)** plus an **external censor**, and
  weighs both the written report and the oral defense. This skill only sees the
  written report, so frame the grade as "based on the written thesis."
- Grading is **absolute against learning objectives**, not curved. The question is
  "how fully does this fulfil the objectives, and how serious are the weaknesses?"
  — which maps naturally onto the issue severities in the review (Critical/Major/
  Minor weaknesses).
- A strong, complete, correct Master's thesis that does what it set out to do,
  with sound method and clear contribution, lands in the **10–12** range. **7** is
  solid work with real but non-fatal weaknesses. Dropping toward **4–02** signals
  significant gaps in rigor, correctness, or completeness.

## Mapping issues to a band (heuristic)

Anchor the band to the issues you actually found and their severity — not to how
the request was phrased. A "find the gaps" prompt and a "give it a fair review"
prompt, applied to the same thesis, should surface the same load-bearing problems
at the same severities and therefore land in the same band. If your band shifts
when the only thing that changed is the tone of the ask, you've let framing leak
into the grade — recheck the severities.

This is a sanity check, not an algorithm — the examiner overrides it:

- Any **unaddressed Critical** issue (a core claim unsupported, a key result
  wrong) caps the band well below 12 and may pull toward 4–7 until resolved.
- Several **Major** issues → 7-ish; a few minor ones with otherwise strong work →
  10; near-clean and genuinely strong → 12.
- Always pair the band with the one or two specific things that would move it up.
